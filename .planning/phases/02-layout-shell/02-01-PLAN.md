Create `src/components/layout/Header.tsx` as a "use client" component.

**Imports needed:**
- `useState` from "react"
- `Link` from "next/link"
- `useScroll`, `useMotionValueEvent` from "motion/react" (NOT framer-motion)
- `clientConfig` from "@/content/client.config"
- `BookingLink` from "@/components/ui"
- `useNav` from "./NavContext"

**NAV_LINKS constant (at module top, above component):**
```typescript
const NAV_LINKS = [
  { href: "/services", label: "Services" },
  { href: "/first-visit", label: "First Visit" },
  { href: "/about", label: "About" },
  { href: "/faq", label: "FAQ" },
  { href: "/contact", label: "Contact" },
] as const;
```

**Scroll detection:**
```typescript
const { scrollY } = useScroll();
const [scrolled, setScrolled] = useState(false);
useMotionValueEvent(scrollY, "change", (latest) => {
  setScrolled(latest > 60);
});
```

**Header structure:**
- `<header>` element with `fixed top-0 left-0 right-0 z-50`
- Background transitions: `scrolled` → `bg-[#FAF3EF] shadow-sm` | !scrolled → `bg-transparent`
- Use `transition-all duration-300` for smooth color change
- Inner container: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- Flex row: `h-16 lg:h-20 flex items-center justify-between`

**Logo (left):**
- `<Link href="/">` with Fraunces text
- CSS class: `font-heading font-semibold text-xl lg:text-2xl transition-colors duration-300`
- Color: `scrolled ? "text-[#2C2C2C]" : "text-white"` (text-brand-dark when scrolled, white when transparent)
- Text: `{clientConfig.name}`

**Desktop nav (center/right, hidden on mobile with `hidden lg:flex`):**
- Map over NAV_LINKS, each as `<Link>` with:
  - Color: `scrolled ? "text-[#2C2C2C] hover:text-[#8BAB8D]" : "text-white/90 hover:text-white"`
  - `transition-colors duration-200 text-sm font-medium`
  - No underline default — underline only on hover via `hover:underline underline-offset-4`

**Book Now CTA (desktop, hidden on mobile with `hidden lg:block`):**
- `<BookingLink variant="header" />` — this renders the honey gold button from Phase 1

**Hamburger button (mobile only, `lg:hidden`):**
- `<button>` with `onClick={() => setIsMobileNavOpen(true)}`
- `aria-label="Open navigation menu"`
- `aria-expanded={isMobileNavOpen}`
- Three `<span>` divs forming the hamburger bars:
  - Each span: `block w-6 h-0.5 transition-all duration-300`
  - Color: `scrolled ? "bg-[#2C2C2C]" : "bg-white"`
  - Container: `flex flex-col gap-1.5 p-2`
- At this stage the button opens the nav (close/X animation is in MobileNav.tsx in Plan 02)
- Min tap target: button must be at least `min-w-[44px] min-h-[44px]` — use `flex items-center justify-center`

Export: `export function Header()` (named export).
