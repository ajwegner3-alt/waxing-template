   - Returns `@type: "Service"` schema with:
     - name: service.name
     - description: service.description
     - url: `${config.siteUrl}/services/${service.slug}`
     - provider: `{ "@type": ["LocalBusiness", "BeautyBusiness"], name: config.name, url: config.siteUrl }`
     - Conditional `offers` block only when `service.price != null`: `{ "@type": "Offer", price: service.price, priceCurrency: "USD" }`
   - Import `Service` type from `@/lib/types`

2. `generateFAQPageSchema(faqs: FAQ[]): Record<string, unknown>`
   - Returns `@type: "FAQPage"` schema with `mainEntity` array
   - Each FAQ mapped to `{ "@type": "Question", name: faq.question, acceptedAnswer: { "@type": "Answer", text: faq.answer } }`
   - Import `FAQ` type from `@/lib/types`

Both functions must include `"@context": "https://schema.org"`.

      position: index + 1,
      name: item.label,
      item: `${clientConfig.siteUrl}${item.href}`,

  };
